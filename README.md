# Hospital Management System (SQL)
# [HospitalERD] (Hospital_Management_System)

## Project Overview
This project is a comprehensive database schema for a **Hospital Management System** designed using Oracle SQL. It manages departments, doctors, patients, appointments, admissions, lab tests, and billing.

## Key Features
* **Relational Design:** Includes multiple tables with Primary and Foreign Keys.
* **Complex Queries:** Features Joins, Subqueries, and Aggregations.
* **Real-world Scenarios:** Covers everything from patient registration to medicine prescriptions and billing.

## How to use
You can run the `Hospital_Management_System.sql` file in any SQL environment (Oracle, MySQL, etc.) to build the database and insert sample data.
